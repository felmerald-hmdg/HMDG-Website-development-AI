# HMDG Website Development AI

An AI-powered toolkit for automating the full website development lifecycle — from planning and design to SEO, marketing, and ongoing support.

---

## Project Overview

This project is structured in 5 phases, each targeting a specific stage of the website development and automation process.

---

## Phases

### Phase 1 — Site Planner
**HMDG Site Planner** is a tool that generates a sitemap and wireframe based on the client's prompt.

#### Subtasks
- [ ] Site Planner Questionnaires
- [ ] Generate Sitemap based on the Questionnaires
- [ ] Generate Wireframe based on the Sitemap
- [ ] Generate Onboarding for Website AI Automation

#### Site Planner Questions
The following information is collected from the client to generate the sitemap and wireframe:

- Client or Business Name
- Business Type
- Business Location
- Showcase
- Services
- Goals
- Features
- Tone and Voice
- Sitemap
- Audience
- Brand Style
- Integrations
- Competitor List
- Competitor Analysis
- User Experience
- Hosting
- Site Language
- Additional Requirements
